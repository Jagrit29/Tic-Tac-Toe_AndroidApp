# Tic-Tac-Toe_AndroidApp
Built Tic Tac Toe Android Application using Android Studio and Java.

## Functionalites:
1. Used Red and yellow color icons for to represent two players.
2. Implmented Animations, so every choice is coming animated from top.
3. All three cases are properly handled.
4. 
<li> Case 1. Player 1 wins
<li> Case 2. Player 2 wins
<li> Case 3. Draw
5. Option to Play Again is implemented.
